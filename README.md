# events-impactus-animation

Landing page for **events.impactuslearning.com/animation**.

- `index.html` — the live animation page (self-contained; images/fonts load from absolute URLs).
- Hosting: the live URL is served by CLDY (cPanel) from an `animation/` folder in the domain document root.
- To deploy: upload `index.html` to the CLDY `animation` folder, or configure cPanel Git Version Control to pull this repo.
